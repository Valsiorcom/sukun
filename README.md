# SUKUN — Vercel + Next.js Ready

This repository has been refactored to a Vercel-friendly Next.js App Router structure while preserving the original UI and brand identity.

Quick start

1. Copy .env.example to .env and fill values.
2. Install dependencies: npm install
3. Run dev server: npm run dev

Main changes

- New directories: app/, components/, lib/, hooks/, types/, styles/
- One component per file and clearer import paths
- New Next.js + Tailwind config for Vercel import

Migration notes and steps are included in MIGRATION_NOTES.md below.
